# Xvert

Convert between CSV, JSON, YAML, and various other formats.

## Installation

```console
$ gem install xvert
```

## Usage

```
$ xvert help
Commands:
  xvert help [COMMAND]  # Describe available commands or one specific command
  xvert jy              # Convert JSON to YAML
  xvert yj              # Convert YAML to JSON
```
